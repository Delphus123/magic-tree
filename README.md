# Magic Tree 🌳

A pixel-art style web game inspired by the classic MSX game "Magical Tree".

## How to Play

Open `index.html` in your browser!

### Controls
- **↑↓** or **W/S** - Move up/down branches
- **←→** or **A/D** - Move left/right
- **Space** or **↑** - Jump
- **Enter** - Pause

### Objective
- Climb the magical tree
- Collect golden fruits (100 pts) and red fruits (50 pts)
- Avoid the birds flying between branches
- Complete 3 levels with increasing difficulty

### Level Goals
| Level | Target Score | Time | Enemy Speed |
|-------|-------------|------|-------------|
| 1 | 1,000 | 60s | Slow |
| 2 | 2,500 | 75s | Medium |
| 3 | 4,000 | 90s | Fast |

## Features
- 🎮 Pixel art graphics
- 🔊 Modern sound effects
- ⭐ 3 levels with accurate scoring
- 🌙 Beautiful night sky atmosphere
- 📱 Responsive design

## Tech Stack
- Pure HTML5 Canvas
- JavaScript (no frameworks)
- Web Audio API for sound
